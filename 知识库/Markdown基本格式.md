## 强调与行内格式

- 用两个星号包住是 \*\***加粗**\*\*。
- 用一个星号包住是 \**斜体*\*。
- 用反引号包住是 \``行内代码`\`。
- 用两个波浪号包住是 \~\~~~删除线~~\~\~。

## 代码块

\`\`\`python
```python
def hello():
    print("Hello, Markdown!")
    # 三个反引号围起来，后面写上语言名就有语法高亮
```
\`\`\`

上面是围栏式代码块。如果是行内，用单个反引号，比如 \``print("hello")`\`。
